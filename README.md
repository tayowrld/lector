# Проект настроен чисто для личного пользования группой 436 РГРТУ, не является коммерческим продуктом.

## Запуск проекта

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите сервер:
   ```bash
   node server/server.js
   ```

3. Запустите фронтенд:
   ```bash
   npm run dev
   ```
